## Q3. For users in Mexico, which system did KAST integrate to allow instant MXN deposits and withdrawals?

| 選択肢 | 正誤 |
|--------|------|
| PIX | :ErrorCross: |
| ACH | :ErrorCross: |
| **SPEI** | :Checkmark: |
| SEPA | :ErrorCross: |

### 解説
KASTはメキシコ中央銀行が運営する銀行間送金ネットワーク「SPEI（Sistema de Pagos Electrónicos Interbancarios）」を統合しました。これにより、KASTアプリ内でメキシコペソ（MXN）の直接送受信が可能になっています。メキシコの銀行口座からペソを入金するとリアルタイムでUSDに自動変換され、逆にUSDをペソとしてSPEI経由で出金することも可能です。BBVA、Banorte、Santander、Nuなどの主要メキシコ銀行に対応しています。

- **PIX** はブラジルの即時決済システム
- **ACH** は米国の電子送金システム
- **SEPA** は欧州の決済システム
